
# 💻 Job Rush - Frontend

This is the client-side application for the Job Rush Recruitment Platform. It is built with React, Vite, and Tailwind CSS 4, focusing on a premium, responsive user interface.

Original Design: [Figma Design Link](https://www.figma.com/design/uy71aXMRyJl66yP2DeStLd/JobRush-Job-Portal-Design)

## 🛠️ Tech Stack

- **Framework**: React 18 (Vite)
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Data Viz**: Recharts
- **Components**: Shadcn UI / Radix UI

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Environment Configuration**:
   Create a `.env` file and set the backend API URL:
   ```env
   VITE_API_URL=http://localhost:5001
   ```

3. **Run Development Server**:
   ```bash
   npm run dev
   ```

## 📂 Structure

- `src/components`: UI primitives and layout components.
- `src/pages`: Top-level views (Landing, Dashboard, Auth, etc.).
- `src/context`: Global state management for Auth and Theme.
- `src/assets`: Static images and global styles.
